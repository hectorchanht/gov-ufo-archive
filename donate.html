Help keep the archive independent.

realufo.org runs on volunteer time, GitHub Pages bandwidth, and an occasional VPS for the Akamai-fronted scrapers. No advertising networks. No tracking. No paywalls. No LLM-bait. If the archive saved you a research hour, kick a little back.

The pledge

We will never sell ad space. We will never set a tracking cookie. We will never paywall a primary-source government record. If we ever have to, we'll shut the site down first and tell you why.

Non-cash ways to help

Money is not the only useful thing.

  • File a FOIA / SAR / LAI request. Get a previously un-released record into a national archive; we'll mirror it.
  • Spot a dead source URL. Open an issue — /stats.html shows the current dead-link rate.
  • Add a new national archive. See CONTRIBUTING.md.
  • Translate a case page into the source-jurisdiction language (French for GEIPAN, Portuguese for FAB, Spanish for SEFAA/CEFAe…).
  • Cite the archive. Academic papers, podcasts, journalism — link to specific records, not the homepage.

What we do not accept

No paid placements, no sponsored content, no logos in the page chrome, no "we paid you so write nicely about us" arrangements. The editorial line is verbatim source-agency text, full stop.

Specific declines: no AdSense / Ezoic / Mediavine / Outbrain / Taboola (kills CWV and the no-cookies pledge), no Patreon (their tier-locked content model doesn't fit a public archive). We accept Bitcoin via Lightning because the donor stays anonymous and the platform fee is zero.
