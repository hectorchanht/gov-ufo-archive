|----------|-------------|
| client→SW | Browser registers /sw.js; SW intercepts all navigation + asset fetches |
| SW→R2 | SW caches images/fonts from assets.realufo.org (cross-origin) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-09 | Tampering | SW serving stale/malicious cached assets across deploys | mitigate | CACHE_PREFIX includes COMMIT_SHA per D-22 + SW-06; activate handler deletes all `realufo-v<old-sha>-*` caches before claim; updateViaCache: 'none' on registration |
| T-04-10 | Spoofing | SW handler in cached HTML overrides new deploy's SW | mitigate | _headers `/sw.js` returns `cache-control: no-cache, no-store, must-revalidate` (Phase 1 PMS-02 invariant); ALLOW_SKIP_WAITING=false on initial Phase 4 deploys (Pitfall #4) |
| T-04-11 | Information Disclosure | SW caches authenticated routes | mitigate | denylist on NavigationRoute excludes /admin, /_, /api; no auth surfaces exist in read-only archive (PROJECT.md) |
| T-04-12 | Repudiation | Browser disagrees with server on cache version after deploy | mitigate | updateViaCache: 'none' forces fresh /sw.js check on every navigation; CACHE_PREFIX rotation ensures clean transition |
| T-04-13 | Denial of Service | Precache > storage quota (chrome default ~10% disk) | mitigate | maximumFileSizeToCacheInBytes: 5 MB cap; ExpirationPlugin maxEntries: 500 on images; PDFs/videos NEVER precached (size-prohibitive per SW-04) |
| T-04-14 | Tampering | Cross-origin opaque response cache poisoning (R2) | mitigate | CacheableResponsePlugin({statuses:[0,200]}) on image route — opaque responses cached but only with valid status; CLAUDE.md §11 forbids `crossorigin="anonymous"` on `<video>` (CF playback) — accept opaque for videos but they hit NetworkOnly route anyway |
| T-04-SC | Tampering | npm supply-chain for @vite-pwa/astro + workbox-* | mitigate | Pin minor versions in package.json; pnpm-lock.yaml integrity checks; @vite-pwa/astro is the official Vite PWA Astro adapter; workbox-* are official Google Chrome team packages |
</threat_model>
