## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| `data/wargov.json` row text → rendered HTML | Astro templating; built-in `{value}` escapes for free; raw `set:html` only used for shards manifest JSON (already encoded) |
| Lazy-loaded shard `data/wargov-shard-*.json` → `grid.insertAdjacentHTML` | Client-side templating — text values MUST go through HTML escaping (`esc()` in the inline script) before insertion |
| User-controlled URL hash → IntersectionObserver disable | URL hash anchors trigger native scroll-to-anchor; should still work for `#card-<slug>` after lazy-load completes |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-15 | XSS | Card.astro `{row.Title}` if title contains `<script>` | mitigate | Astro auto-escapes `{expr}` text (Astro docs: "All values used as text are HTML-escaped by default"). NO `set:html` on row fields. Verified by reading Astro 5 templating docs. |
| T-03-16 | XSS | Client-side `templateCard()` in lazy-load script using string concatenation | mitigate | `esc()` helper applies HTML entity escaping to all row values before `${row.X}` interpolation; uses the same character set as the existing scripts/templates/archive.py esc function |
| T-03-17 | Tampering | A poisoned shard JSON (e.g. modified at edge CDN) could inject HTML via `templateCard` | mitigate | _headers (Phase 2 02-01) Cache-Control on JSON shards is the default `/*` rule (`X-Content-Type-Options: nosniff` covers MIME confusion); CSP deferred to Phase 6 per D-07 means script-src can be tightened later |
| T-03-18 | Tampering | `#card-<slug>` slugify function drifts from `scripts/snapshot-urls.py` → URL-CONTRACT.txt breakage | mitigate | Plan 03-06 acceptance criterion runs `python3 scripts/snapshot-urls.py --check` against the preview URL — drift fails the gate |
| T-03-19 | Information Disclosure | OG image leak via uncontrolled `row['Modal Image']` | accept | All images are public-domain government media (CLAUDE.md §9) |
