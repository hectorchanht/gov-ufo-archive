**2. [Rule 1 — Bug] Comma in static header comment trips `grep -q ','` verify clause**

- **Found during:** Task 1 sanity-check (`! grep -q ',' _redirects`)
- **Issue:** Initial header line `# Status codes: 200 = rewrite (URL stays), 301 = permanent redirect.` contained a literal comma.
