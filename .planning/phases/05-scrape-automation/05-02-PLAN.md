| T-05-02-01 | Spoofing | UA spoofing on outgoing fetch | accept | The realistic Chrome UA is intentional ‚ÄĒ measuring the same blocking surface a production cron Worker would see. Akamai sees the same UA in production, so this is a real measurement, not a deception. |
