# realufo.org is a static archive of public-domain government UAP records.
# We are particularly interested in:
#   - URL-rewrite or XSS issues in the search/timeline/map pages
#   - Source-URL integrity (e.g. a manifest entry pointing at a malicious file)
#   - Service-worker cache-poisoning vectors
# We are not running:
#   - User authentication, accounts, or session state
#   - A backend, database, or any server-side code
#   - Vercel, AWS, GCP, or paid third-party services
# Please give us a reasonable disclosure window before public release.
