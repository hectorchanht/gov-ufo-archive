/* TOOLING */
  Static HTML — no build step, no framework
  Python 3 — Standard library + curl_cffi (Akamai TLS impersonation)
  Wayback Machine — for AARO pages lost to Akamai blocks
  Leaflet — open-source map tiles, no API key
  OpenStreetMap — tile data, © contributors
  Umami Cloud — cookieless, privacy-friendly analytics (free tier)
  GitHub Pages — hosting
  GitHub Releases — payload distribution for files > 100 MB
